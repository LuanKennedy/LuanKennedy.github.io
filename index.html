Framework, o qual tenho familiariedade em razão de projetos desenvolvidos, atrelado com a minha proeficiência no frontend e correlacionado com a minha anterior experiência profissional, onde a qual, eu utilizava, para adicionar features e corrigir bugs.

Javascript, CSS and HTML

Tríade da programação web, onde o meu conhecimento está de intermediário para avançado, e o javaScript é linguagem à qual fundamenta minha lógica de programação, devido ela ser a base dos frameworks, Libs e até mesmo banco de dados não relacionais, como o MongoDB(sendo possível fazer query), dos quais faço uso de maneira exuberante, a principal atualização da linguagem que eu me aprimorei, foi o ES6, uma maneira mais performática de programar em JS
