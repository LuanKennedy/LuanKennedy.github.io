Conhecimentos

Java

Nessa linguagem de programação, tenho como principal valência a programação orientada à objetos, e tratativas de exception com try catch, além de ter um breve conhecimento no que se diz respeito ao garbage collector, estou buscando me desenvolver na linguagem, tendo em vista que a curva de aprendizado é mais lenta, devido ser uma linguagem verbosa.

React & React Native

Framework ou Lib, cujo qual tenho muita familiariedade em detrimento de projetos desenvolvidos, atrelado com a minha proeficiência no frontend e intrinsecamente correlacionado com a minha anterior experiência profissional, onde a qual, eu utilizava dessa ferramenta, para adicionar features e corrigir bugs.

Javascript

Linguagem à qual fundamenta minha lógica de programação, devido ela ser a base dos frameworks, Libs e até mesmo banco de dados não relacionais, como o MongoDB(sendo possível fazer query), dos quais faço uso de maneira exuberante, a principal atualização da linguagem que eu me aprimorei, foi o ES6, uma maneira mais performática de programar em JS

Node

Esse motor que roda no backend das minhas aplicações, é responsável, pelas requisições, respostas, tratativas, etc. Me auxilia com bastante frequência, por ser um excelente parceiro ao react e trazer mais desempenho para as aplicações.

Boostrap & CSS

Dois aspectos fundamentais dos quais utilizo, pelo fato de regerem esse novo conceito de mobile first, ocasionando em sites responsivéis, ou seja, adaptativos em dispositivos distintos, fornecendo uma UI/UX ideal para o usuário

Git e GitHub

Importantes ferramentas em instituições que utilizam metodologia agile, versionamento de código, code review, etc. Possuo conhecimento nesse segmento, além dessas ferramentas, já utilizei o JIRA e BitBucket, as quais têm o mesmo intuito, porém fornecem mais recursos para os time de desenvolvimento, do developer ao CTO.
